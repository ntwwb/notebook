1.git命令查看、创建、切换、删除分支
    git tag 
    git branch
    git branch -b 分支名
    git clone 库url
    git checkout -b OSTACK-296
    git clone 库url -b OSTACK-296
    1). 查看远程分支
        $ git branch -a 
    2). 查看本地分支
        $ git branch  
    3). 创建分支
        $ git branch test
        $ git branch     (查看是否创建成功)
    4). 切换分支到test
        $ git checkout test  
    5). 删除本地分支
        $ git branch -d test 
    6). 查看本地和远程分支  -a。前面带*号的代表你当前工作目录所处的分支
        $ git branch -a
